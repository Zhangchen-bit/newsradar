LLM API 配置

你的 key 只存在你浏览器的 localStorage，从不上传到任何服务器。 调用直接从你浏览器发到 LLM 服务商。

使用建议
  • Anthropic 需要在 Console 开启 anthropic-dangerous-direct-browser-access；推荐 claude-sonnet-4-5 或 claude-3-5-sonnet-20241022。
  • OpenAI 兼容 任何兼容 /v1/chat/completions 的端点都可。DeepSeek 例：base_url=https://api.deepseek.com，model=deepseek-chat。
  • 单次摘要约消耗 8-15K input + 2-3K output token。
